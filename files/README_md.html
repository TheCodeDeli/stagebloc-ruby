StageBloc - Ruby API Wrapper

The StageBloc gem is an API wrapper to give push/receive data for an authenticated user.

For authentication via OAuth check out our OmniAuth Strategy for Stagebloc.

Installation

Add this line to your application's Gemfile:

gem 'stagebloc'

And then execute:

$ bundle

Or install it yourself as:

$ gem install stagebloc
